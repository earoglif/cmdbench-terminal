## cmdbench-terminal

Десктопное приложение-терминал на базе Tauri с поддержкой множественных вкладок и системы управления командами. Позволяет создавать, организовывать и быстро выполнять команды с параметрами, синхронизировать их и настраивать интерфейс под свои нужды.

### Основные возможности

- **Множественные терминалы** — работа с несколькими вкладками терминала одновременно
- **Управление командами** — создание, редактирование и группировка команд с параметрами
- **Синхронизация с облаком** — сохранение команд и групп команд в облаке для доступа с разных устройств
- **Настройки** — кастомизация внешнего вида, горячих клавиш и профилей оболочки
- **Локализация** — поддержка русского и английского языков

### Горячие клавиши

- `Ctrl+T` — новый таб
- `Ctrl+W` — закрыть текущий таб
- `Ctrl+Tab` — следующий таб
- `Ctrl+Shift+Tab` — предыдущий таб
- `Ctrl+1-9` — переключение на таб по номеру
