Возможности

Откройте для себя все мощные возможности CMDBench Terminal

Шаблоны команд

Создавайте, храните и запускайте команды как шаблоны с полями (string, number, file, directory, select). Для file и directory открывается проводник, чтобы выбрать путь.

Группы команд

Собирайте сохранённые команды в группы и держите библиотеку в порядке.

Меню и горячие клавиши

Запускайте команды из выпадающего списка или назначайте сочетания клавиш в настройках.

Импорт и экспорт

Переносите команды и группы между устройствами через JSON-файлы.

Настройка интерфейса

Темы, шрифты и параметры терминала под ваш стиль.

React + Tauri

Лёгкое кроссплатформенное приложение для Windows, macOS и Linux.
